# 부분집합 문제
- 유한 개의 정수로 이루어진 집합의 부분집합 중
그 집합의 원소를 모두 더한 값이 0이 되는 경우가 있는지 알아내는 문제

1. 부분집합의 수 구하기
- 집합의 원소가 n개일 때 공집합을 포함한 부분집합의 개수는 2^n
- 이는 각 원소를 부분 집합에 포함 시키거나 포함시키지 않는 2가지의 경우를 모든 원소에 적용한 경우의 수와 같다.

2. 부분집합의 bit 생성
```py
a = [2, 3, 7]
bit = [0] * 3
for i in range(2):
    bit[0] = i
    for j in range(2):
        bit[1] = j
        for p in range(2):
            bit[2] = p
            # print(bit)
            for b in range(3):
                if bit[b]:
                    print(a[b], end=' ')
            print(bit)
'''
[0, 0, 0]
7 [0, 0, 1]
3 [0, 1, 0]
3 7 [0, 1, 1]
2 [1, 0, 0]
2 7 [1, 0, 1]
2 3 [1, 1, 0]
2 3 7 [1, 1, 1]
'''
```
